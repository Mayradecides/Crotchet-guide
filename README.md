# Crochet Stitch Counter

A Python project to help track crochet stitches completed in a project.

## How to run

1. Make sure you have Python 3.x installed
2. Download or clone this repository.
3. Open a terminal, go to the project folder, and run:

   ```
   python main.py
   ```

That's it! Follow the on-screen instructions to add or review your stitch counts.
